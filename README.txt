Enable Googlebots to Render Your Site

== Description ==

This Prerender Plugin integrates with the https://prerender.io service.

Once you have signed up to Prerender.io and obtained a token, you can save that token and activate the Prerender Plugin.

Our plugin will automatically add all required code to the .htaccess file for WordPress and a metatag on all public pages to allow Googlebot and other crawlers to perfectly crawl your website.

[https://hekkup.com](https://hekkup.com)

== Installation ==

To install this plugin:

1. Make a backup!
2. Download the plugin
3. Upload the plugin to the wp-content/plugins directory,
4. Go to “plugins” in your WordPress admin, then click activate.
